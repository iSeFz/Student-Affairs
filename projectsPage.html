Student's Affairs

This website is responsible for managing data related to students registered to the university.

It allows student affairs' personnel to:
  • Add a new student to the database.
  • Update an existing student information.
  • Delete an existing student or make them inactive.
  • Search and view all students.

Laboratory Tracking

This website is responsible for managing laboratory information.

It allows managers to:
  • Add a new laboratory to the database.
  • Update an existing laboratory's information.
  • Delete an existing laboratory's information.
  • Search and view all laboratories.
  • Report problems in a specific laboratory (if they exist).

Higher Studies

This website is responsible for managing higher studies students information.
It allows users to:
  • Add a new student to the database.
  • Update an existing student information.
  • Add a new course and link it to a department.
  • Delete an existing student or make them inactive.
  • Search and view all students.
  • Register courses for a specific student.

Human Resources

This website is responsible for managing employee information.

It allows HR personnel to:
  • Add a new employee to the database.
  • Update an existing employee's information.
  • Delete an existing employee's information.
  • Search and view all employees.
  • Submit a vacation form for an employee.
